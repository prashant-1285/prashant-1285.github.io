Academic Projects

Academic Projects

Here are some of the projects I've worked on during my academic journey.

Colorimetric camera characterization using Neural Network and SVM regression.

Norwegian University of Science and Technology, Norway

This project aimed to use machine learning models like SVM Regression with different kernels and ANN with different numbers of hidden neurons to create a camera characterization model using the color chart. Then it is experimented on some of the test color patches and textile samples. Although it is essential to consider that the color patches used to build the model and the target must be of the same material, this project tries to find if the SVM regression and ANN are able to model both the conditions i.e. when the target sample is same and different.

Keywords: Digital camera Characterization, SVM, Neural Network

3D Reconstruction of a static scene from Multiview

University Jean Monnet, France

Created three-dimension model of a scene from two-dimensional images. 3 go pro cameras were used to take image of the static scene from three direction. The cameras were calibrated using the ArUco Boards. The necessary intrinsic and extrinsic parameters were computed, performed image rectification, calculated the disparity map and the projection matrix. Finally, the reconstruction was performed in mesh lab from the generated point clouds.

Keywords: Aurco Checkboard, Camera Calibration,Image Rectification, DepthMaps

Latent space study for Color Constancy Using Variational Autoencoders

University Jean Monnet, France

Explored manipulating the latent space using Variational Auto Encoders to manipulate the colors for the purposes of color correction on INTEL-TAU dataset. Found that, there seems to be certain dimensions in the latent space that are correlated to the different color features. Even when trained on the model, this color information did show up even when retraining the model with different latent space size, leaving us to believe that the color information can be extracted any time. A strong verification of this statement is to be done as a future work

Keywords: AutoEncoder, LatentSpace, ColorConstancy

Particle Swarm Optimization for Illumination Optimization for Improved Color Contrast

University of Eastern Finland, Finland

The project focused about the approach to enhance color perception by using the Particle Swarm Optimization to design optimal illuminations using the LEDs. The study explored the use of PSO to create illuminations that minimize or maximize the color difference between two selected points on the test chart whose spectral image was acquired by linescanner (Hyperspectral Camera).The optimized illuminations convex combinations of 10 LED illuminants, effectively spanning the specified wavelength range. The effectiveness of these optimal illuminations was verified, utilizing an LED-based spectrally tunable light source. Along with the computational results, visual simulations on the color blindness test pages was also done using the LED light source. These LED based illuminants could help improve contrast on images

Keywords: Particle Swarm Optimization, Hyperspectral Imaging,Color Contrast

Hyperspectral Imaging Acquisition, Image correction, Segmentation

University of Eastern Finland, Finland

The project was implemented to get understanding of Hyperspectral Imaging Acquisition, perform necessary white correction and apply some segmentation technique for particular materials. Images from different plastic and non plastic green materials using Hyperspectral devices were acquired like IR, Tunable Camera, Nuance Camera, and Specim Scanner. All the necessary white correction was performed for each type of camera using specific ways required. Spectral Angular mapping, was the segmentation technique employed. A particular area of interest was selected from each material, and it was compared using SAM with the entire spectral cube to create a segmentation mask that was ultimately superimposed. For various materials and cameras, different thresholds were needed. Thus, cutoff points were modified.

Keywords: White Correction,Dark Correction, Hyperspectral Imaging, Spectral Image Segmentation
